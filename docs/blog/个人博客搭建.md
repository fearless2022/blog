## 使用工具

> 一个神奇的文档网站生成器：[docsify](https://docsify.js.org/#/zh-cn/)

## 安装doocsify

* ```bash
  npm i docsify-cli -g
  ```

## 初始化项目

* ```bash
  # docs：项目目录
  docsify init ./docs
  ```

## 项目结构

* `index.html` 入口文件
* `README.md` 会做为主页内容渲染
* `.nojekyll` 用于阻止 GitHub Pages 忽略掉下划线开头的文件
* 更多自定义内容，查阅官网：[docsify](https://docsify.js.org/#/zh-cn/)

## 本地预览

* ```bash
  # docs：项目目录
  docsify serve docs
  ```
